# README

## Development Language

* Ruby 2.6.5
* Rails 5.2.6

## Employment Term Technologies

* Devise
* Comment Function using Ajax

## Technologies outside curriculum

* Cloudinary
* Weglot

## Execution Procedure

```
$ git clone git@github.com:yakid-japon/isma-dive.git
$ cd isma-dive
$ bundle install
$ rails db:create
$ rails db:migrate
$ rails db:seed
$ rails s
```
